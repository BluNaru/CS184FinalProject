CS 184 Final: Milestone

Team 52: James Fang, Lea Nakatsukasa, Nicholas Pels



Our project is to generate an interactive solar system with unique planets. Like a real solar system, our simulation would include a star, rocky planets, and gaseous planets, with the planets orbiting around the star. The user can interact with specific planets to explore the solar system.

So far, our group has an editable model of a rocky planet. The planet’s colors, size, and terrain can be edited manually. We also have a working orbit for the planets in the solar system. In the planets’ orbit, the user can zoom in on a specific planet by selecting it from a dropdown menu. Gaseous planets are currently in progress, which when added, should be customizable similarly to the rocky planets. There will be an option to add rings to gaseous planets, but we are still in the process of debugging the rings’ appearance.

As of now, we still need to make sure all our components work together seamlessly. We only recently started incorporating the rocky planets into the orbit demo. For gaseous planets, we need to debug the implementation of rings and apply a striped, gaseous texture to the planet itself so it is not a flat sphere.
